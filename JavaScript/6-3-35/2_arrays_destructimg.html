Rest

when we assign the variables to each item of an array ,it is difficult to assign each item one by one so we use (...rest) to print the remaining item of an array,see the example below in array2

when we want to leave the middle item we just use (var1, , var2), it will print the 1st and 3rd item...similarly (var1, var2, ,) it will print the 1st and 2nd item and will not print the third item(see example below array3)
